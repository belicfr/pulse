# Pulse project

## Rules

### Variable definition:
Existing types:
- string (between " or ')
- integer
- boolean
```text
name = value
```

### Function definition:
**Warning:** only function header can be defined,
function body definition is **WIP**.
```text
function name
function name2 -> arg1, arg2
```

### Console output:
Possible values:
- Native types
- Variables
```text
print value
```
