# Pulse project

## Rules

### Variable definition:
Existing types:
- string (between " or ')
- integer
- boolean (not expression)
```text
firstname = "John"
lastname = 'Doe'
age = 20
isAdult = true
```

### Function definition:
**Warning:** parameter definition is WIP, it cannot be used.
```text
function name
    ...
    
function name2 -> arg1, arg2
    ...
```

### Function call:
**Warning:** none parameter can be given during function call. 
Argument usage is **WIP**.
```text
:functionName
```

### Console output:
Possible values:
- Native types
- Variables
```text
print value
```
